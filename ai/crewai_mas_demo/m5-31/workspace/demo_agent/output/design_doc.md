## 需求摘要
实现一个短链接服务，用户可以输入长网址并获取对应的短网址。

## 方案概述
- **技术选型**：使用 Python Flask 作为后端框架。
- **核心模块划分**：
  - 短网址生成模块：负责生成唯一的短网址。
  - 长网址存储模块：存储长网址和对应的短网址映射关系。
  - 用户接口模块：提供用户提交长网址并获取短网址的服务。
- **数据流简述**：用户通过 HTTP POST 请求提交长网址，服务端生成短网址并返回给用户。

## 接口设计
- **接口**：`POST /shorten`
- **路径**：`/shorten`
- **方法**：`POST`
- **入参**：
  ```json
  {
    "long_url": "https://example.com"
  }
  ```
- **出参**：
  ```json
  {
    "short_url": "http://short.url/abc123"
  }
  ```

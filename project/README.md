### 大文件的分片上传流程可以通过以下几个步骤实现
- 在前端，将上传大文件拆分成一片又一片，并标记分片顺序和文件hash；
- 前端发起上传请求，每一次请求给后端带一片文件，在后端接受分片的所有文件并缓存；
- 当所有分片文件都上传完，再通过前端发出合并请求，告知后端将分片的文件合并；

 -前端如何确定文件的id，如何才能得到文件的唯一标识？此处我们需要依赖一个插件 spark-md5
